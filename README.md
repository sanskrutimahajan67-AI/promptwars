# promptwars
🌱 AI Carbon Footprint Agent tailored for Indian lifestyles. Powered by FastAPI &amp; Gemini 2.0, CarbonSaathi calculates monthly CO2, compares it to India's average, awards eco-badges, and delivers localized sustainability tips. 🌍🇮🇳
## 🎯 Problem
Indians lack personalized, India-specific 
carbon footprint awareness tools.
Generic calculators don't understand 
Indian lifestyle, cities, or habits.

## 💡 Solution
CarbonSaathi is an AI Agent that:
- 🏙️ Understands your Indian city & lifestyle
- 📊 Calculates your monthly CO2 in kg
- 🇮🇳 Compares with India average (165kg/month)
- 🎯 Gives city-specific personalized tips
- 🏆 Awards badges based on your footprint

## 🏅 Badges
| Badge | Meaning |
|---|---|
| 🌱 Seedling | Above average footprint |
| 🌿 Sapling | Near average footprint |
| 🌳 Tree | Below average footprint |
| 🌲 Forest Guardian | Very low footprint |

## 🛠️ Tech Stack
- *AI* → Google Gemini API (gemini-2.0-flash)
- *Frontend* → HTML, CSS, JavaScript
- *Backend* → FastAPI (Python)
- *Deployment* → Vercel
